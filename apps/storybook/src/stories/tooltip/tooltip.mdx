# Tooltip

The Tooltip component in the "@baka/react" library allows you to display additional information or actions when users hover over or interact with an element.

## Installation

Before using the Tooltip component, make sure you have the "@baka/react" library installed in your project.

```bash
npm install @baka/react
```

## Importing

You can import the Tooltip component like this:

```javascript
import { BakaTooltip } from "@baka/react";
```

## Usage

### Basic Usage

The Tooltip component can be used to wrap any element you want to add additional information or actions to. Here's a basic example:

```jsx
<BakaTooltip>
  <div className={styles["content"]}>
    <BakaLabel variant={"title-small"}>Title</BakaLabel>
    <BakaParagraph variant={"body-medium"}>
      Supporting line text lorem ipsum dolor sit amet, consectetur
    </BakaParagraph>
  </div>
  <div className={styles["actions"]}>
    <BakaButton variant={"text"}>Action</BakaButton>
    <BakaButton variant={"text"}>Action</BakaButton>
  </div>
</BakaTooltip>
```

### Variations

The Tooltip component supports different variations:

- `variant={"plain"}`: Use the "plain" variant for simple text tooltips without additional styling.

```jsx
<BakaTooltip variant={"plain"}>
  <BakaText variant="body-small">Supporting text</BakaText>
</BakaTooltip>
```

### Customization

You can customize the width and style of the Tooltip by applying CSS styles directly to the component.

```jsx
<BakaTooltip variant={"plain"} style={{ width: 184 }}>
  <BakaParagraph variant="body-small">
    Supporting text Body text string goes here psum dolor sit amet, consectetur adipiscing elit, sed
    do eiusmod tempor incididunt
  </BakaParagraph>
</BakaTooltip>
```
