## Navigation

The `Navigation` component is used to create a navigation bar typically placed at the top, bottom or the side of an app. It allows users to switch between different sections or views within the app.

<Demo src="/demos/material-you/navigation/" style={{ height: 600 }} />

### Usage

```jsx
import { BakaBottomNavigation, BakaNavigationItem, BakaIcon, BakaLabel, BakaBadge } from "baka-ui";

// Inside your component
<BakaNavigation>
  <BakaNavigationItem selected={true}>
    <BakaIcon variant={"filled"}>fiber_manual_record</BakaIcon>
    <BakaLabel>BakaLabel</BakaLabel>
  </BakaNavigationItem>
</BakaNavigation>;
```
