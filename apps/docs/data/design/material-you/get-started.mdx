# Get Started

## Installation

```bash
yarn add baka-material-you
```

## Usage

The `baka-material-you` package provides both a `css` and [`scss`](https://sass-lang.com/) distribution of the styles.

- CSS

   ```jsx
   import "baka-material-you/dist/index.css";
   ```

- SCSS

   ```scss
   @use "baka-material-you";
   ```

## Fonts & Icons

The `baka-material-you` package does not include the fonts and icons required for the Material You design system. You can find the fonts [here](https://fonts.google.com/specimen/Roboto?query=Roboto) and the icons [here](https://fonts.google.com/icons).

To reference the fonts and icons, you can use the following `link` tags:

```html
<link
  href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap"
  rel="stylesheet"
/>
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200"
/>
```
