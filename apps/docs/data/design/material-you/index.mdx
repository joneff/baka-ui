# Overview

The `baka-material-you` package represents the Google's [Material You](https://m3.material.io/) design system implemented through the [`baka-ui`](https://www.npmjs.com/package/baka-ui) primitives.

It distributes a collection of all the components and styles that are part of the Material You design system.

## Design

The design was created by Google and is available at [Material You](https://m3.material.io/). For more information about the design system, please refer to the [official documentation](https://m3.material.io/).
